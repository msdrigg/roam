# 手动添加电视

1. 查找您的电视 IP 地址
    - 打开电视并导航至 **设置** > **网络** > **关于**
    - IP 地址应如 10.x.x.x，172.x.x.x，173.x.x.x 或 192.168.x.x 形式
    - 此页面可能列出一个"网关"地址和一个"IP地址"。请确保您不使用"网关"地址
2. 导航至 Roam 设置然后点击“手动添加设备”
3. 随您喜欢命名设备，然后输入 Roku 电视显示的设备 IP，无需做任何改动
4. 点击保存。现在您的 Roku 应能连接并正常工作

## 如果你手动添加电视后，Roam 还是无法连接怎么办？

如果 Roam 仍无法控制您的 Roku， 请尝试以下步骤

-   确保您的 iOS 设备连接的是与 Roku 电视相同的 WiFi 网络
-   确保您的电视已打开
-   确保已为 Roam 启用本地网络权限（如果已启用则禁用并重新启用）
    - 在 macOS 上：转至系统设置 -> 隐私及安全 -> 本地网络 -> Roam
    - 在 iOS 上：转至设置 -> 应用程序 -> Roam -> 本地网络
-   在此处查看更多可能性 [https://support.roku.com/article/115001480188](https://support.roku.com/article/115001480188)
