# 手动添加电视

1. 找到您的电视的 IP 地址
    - 打开您的电视，并导航到**设置** > **网络** > **关于**
    - IP 地址应像 10.x.x.x、172.x.x.x、173.x.x.x 或 192.168.x.x 这样
    - 这个页面可能列出一个 "网关" 地址和一个 "IP 地址"。确保你不使用 "网关" 地址
2. 导航到 Roam 设置并点击 "手动添加设备"
3. 根据你想要的方式命名你的设备，并准确地输入 Roku 电视上显示的设备 IP
4. 点击保存。现在你的 Roku 应该能够连接并正常工作

## 如果你手动添加了电视，Roam 还是不能连接怎么办？

如果 Roam 仍然无法控制你的 Roku，请尝试以下步骤：

-    确保您的 iOS 设备连接到与您的 Roku 电视相同的 WiFi 网络
-    确保您的电视已经打开
-    确保已为 Roam 启用了本地网络权限（如果已启用，请禁用并重新启用）
     -   在 macOS 上：转到系统设置 -> 隐私和安全 -> 本地网络 -> Roam
     -   在 iOS 上：转到设置 -> 应用 -> Roam -> 本地网络
-   在这里查看更多可能性 [https://support.roku.com/article/115001480188](https://support.roku.com/article/115001480188)

## 如果我有复杂的网络/VPN 设置呢？这个应用使用什么协议？

-   Roam 使用两种不同的协议与电视通信
    -   TCP（HTTP/Websockets）在 8060 端口向电视发送命令
    -   WOL 魔法封包（向地址 255.255.255.255 的 UDP 组播）以从深度睡眠状态唤醒电视
-   所有 Roku 电视都使用 8060 端口，电视端无法更改此设置。但是，如果你有某种端口转发设置并想从 Roam 使用不同的出站端口，这是可能的。你只需要在 "Ip 地址" 字段中输入 `<IP>:<Port>`，而不仅仅是 `<IP>`。例如，输入 `192.168.8.242:8061`，将使用选定的端口。
