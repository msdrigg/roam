# 手动添加电视

1. 查找您电视的 IP 地址
    - 打开电视，进入 **设置** > **网络** > **关于**
    - 如果您没有实体遥控器或其他方式来控制电视，请在家用路由器的管理界面或 DHCP 客户端列表中查找 Roku 的 IP 地址
    - IP 地址通常类似于 10.x.x.x、172.x.x.x、173.x.x.x 或 192.168.x.x
    - 该页面可能显示“网关”地址和“IP 地址”。请确保您没有使用“网关”地址
2. 打开 Roam 设置并点击“手动添加设备”
3. 设备名称可自行设定，并请严格按照 Roku 电视上显示的 IP 填写设备 IP
4. 点击保存。现在您的 Roku 应该可以正常连接和工作了

## 如果您手动添加电视后 Roam 仍无法连接或连接不正常怎么办？

如果 Roam 仍然无法控制您的 Roku，请尝试以下步骤

-   [仅限 WatchOS]：请进入 **设置 -> 系统 -> 高级系统设置 -> 通过移动应用控制**，确保设为 **宽松**
-   请确保您的 iOS 设备与 Roku 电视连接在同一个 WiFi 网络下
-   确保您的电视已开机
-   请确保已为 Roam 启用本地网络权限（如果已启用，可以先关闭再重新启用）
    -   在 macOS: 进入系统设置 -> 隐私与安全性 -> 本地网络 -> Roam
    -   在 iOS: 进入设置 -> 应用 -> Roam -> 本地网络
-   如果您的家庭网络配置发生变化，导致之前可用的设备无法使用，请先从 Roam 中删除已保存的设备，再重新扫描添加
-   如果 Roku 没有连接 WiFi 且没有实体遥控器，请按照 Roku 手机 APP 的连接步骤操作：[https://support.roku.com/article/115001480188](https://support.roku.com/article/115001480188)
-   其他可能情况请参阅 [https://support.roku.com/article/115001480188](https://support.roku.com/article/115001480188)

## 如果我有复杂的网络/VPN 设置？此应用使用了哪些协议？

-   Roam 使用多种协议与电视通信
    -   使用 8060 端口（TCP，HTTP/Websockets）向电视发送指令和查询设备状态
    -   使用 WOL 魔术包（UDP 组播，目标地址 255.255.255.255）唤醒深度休眠的电视
    -   RDP（UDP，端口 6970）用于耳机模式下的音频流传输
-   所有 Roku 电视均使用 8060 端口，且在电视端无法更改。如果您使用端口转发，并希望 Roam 端使用不同的外发端口，则可以。在“IP 地址”字段输入 `[IP]:[Port]`，而不是只输入 `[IP]`。例如输入 `192.168.8.242:8061`，端口就会使用 `8061`。
