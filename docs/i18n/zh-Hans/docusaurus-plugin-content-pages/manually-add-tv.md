# 手动添加电视

1. 查找电视的 IP 地址
    - 打开电视，导航到 **设置** > **网络** > **关于**
    - IP 地址通常类似于 10.x.x.x、172.x.x.x、173.x.x.x 或 192.168.x.x
    - 此页面可能会列出“网关地址”和“IP 地址”。请确保你使用的是“IP 地址”，而不是“网关地址”
2. 打开 Roam 设置，点击“手动添加设备”
3. 按你喜欢的方式命名设备，并准确输入 Roku 电视上显示的设备 IP
4. 点击保存。现在 Roam 应该可以连接你的 Roku 并正常运行了

## 如果你手动添加了电视，但 Roam 仍无法连接，或者连接有问题怎么办？

如果 Roam 仍无法控制你的 Roku，请尝试以下操作：

-   [仅限 WatchOS]：请前往 **设置 -> 系统 -> 高级系统设置 -> 通过移动应用控制**，确保设置为 **允许**
-   请确保你的 iOS 设备已连接到和 Roku 电视相同的 WiFi 网络
-   请确保你的电视已开启
-   请确保 Roam 已开启本地网络权限（如果本地网络权限已开启，可以尝试关闭再重新开启）
    -   在 macOS 上：进入系统设置 -> 隐私与安全性 -> 本地网络 -> Roam
    -   在 iOS 上：进入 设置 -> 应用 -> Roam -> 本地网络
-   查看其他可能的原因：[https://support.roku.com/article/115001480188](https://support.roku.com/article/115001480188)

## 如果我的网络环境较为复杂（如使用 VPN 等），或者我想了解该应用用到哪些协议怎么办？

-   Roam 使用多种协议与电视通信
    -   通过 8060 端口的 TCP（HTTP/Websockets）向电视发送指令和获取设备状态
    -   使用 WOL 魔术包（UDP 组播至 255.255.255.255）以便让电视从深度休眠中唤醒
    -   通过 6970 端口的 RDP（UDP）协议用于耳机模式音频流
-   所有 Roku 电视都使用 8060 端口，且无法在电视上更改该端口。如果你有端口转发需求，想从 Roam 端使用不同的出口端口，也是可以的。只需在“IP 地址”栏输入 `[IP]:[Port]`，而不仅仅是 `[IP]`。例如输入 `192.168.8.242:8061`，那端口 `8061` 就会被使用。
