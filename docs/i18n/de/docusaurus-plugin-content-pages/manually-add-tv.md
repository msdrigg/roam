# Einen Fernseher manuell hinzufügen

1. Die IP-Adresse Ihres Fernsehers finden
    - Schalten Sie Ihren Fernseher ein und navigieren Sie zu **Einstellungen** > **Netzwerk** > **Über**
    - Die IP-Adresse sollte wie 10.x.x.x, 172.x.x.x, 173.x.x.x oder 192.168.x.x aussehen
    - Diese Seite könnte eine "Gateway"-Adresse und eine "IP-Adresse" auflisten. Stellen Sie sicher, dass Sie NICHT die "Gateway"-Adresse verwenden
2. Navigieren Sie zu den Roam-Einstellungen und klicken Sie auf "Gerät manuell hinzufügen"
3. Benennen Sie Ihr Gerät, wie Sie möchten, und geben Sie die IP des Geräts genau so ein, wie sie auf dem Roku TV angezeigt wird
4. Klicken Sie auf Speichern. Jetzt sollte Ihr Roku sich verbinden können und normal funktionieren

## Was, wenn Sie den Fernseher manuell hinzufügen und Roam immer noch keine Verbindung herstellen kann?

Wenn Roam Ihr Roku immer noch nicht steuern kann, versuchen Sie bitte die folgenden Schritte

-   Stellen Sie sicher, dass Ihr iOS-Gerät mit demselben WLAN-Netzwerk wie Ihr Roku TV verbunden ist
-   Stellen Sie sicher, dass Ihr Fernseher eingeschaltet ist
-   Stellen Sie sicher, dass die Berechtigungen für das lokale Netzwerk für Roam aktiviert sind (oder deaktivieren und reaktivieren Sie es, wenn es bereits aktiviert ist)
    - Auf macOS: Gehen Sie zu Systemeinstellungen -> Datenschutz und Sicherheit -> Lokales Netzwerk -> Roam
    - Auf iOS: Gehen Sie zu Einstellungen -> Apps -> Roam -> Lokales Netzwerk
-   Sehen Sie weitere Möglichkeiten hier [https://support.roku.com/article/115001480188](https://support.roku.com/article/115001480188)
