# Schauen Sie sich den App Clip an

Möchten Sie die App nicht aus dem App Store herunterladen?

-   Laden Sie den App Clip hier herunter, indem Sie diesen Link auf einem iPhone oder iPad in Safari öffnen und auf das Banner am oberen Rand Ihres Bildschirms klicken

## Sie sehen das "Open App Clip" -Banner nicht?

-   **Sie müssen diesen Link in Safari öffnen, damit er funktioniert**
-   Diese Funktion funktioniert nur auf einem iPhone oder iPad
