## Roam ਬਾਰੇ

Roam ਤੁਹਾਨੂੰ ਸਾਰੀਆਂ ਚਾਹੀਆਂ ਚੀਜ਼ਾਂ ਦਿੰਦਾ ਹੈ ਅਤੇ ਨਾ-ਚਾਹੀਆਂ ਨਹੀਂ

-   Mac, iPhone, iPad, Apple Watch, Vision Pro ਜਾਂ Apple TV 'ਤੇ ਚੱਲਦਾ ਹੈ!
-   Mac 'ਤੇ ਸ਼ਾਰਟਕੱਟ ਤੇ iOS 'ਤੇ ਟੀ.ਵੀ. ਵਾਲੀਅਮ ਕੰਟਰੋਲ ਲਈ ਹਾਰਡਵੇਅਰ ਵਾਲੀਅਮ ਬਟਨ ਵਰਤਣ ਸਮੇਤ, ਸਮਾਰਟ ਪਲੇਟਫਾਰਮ ਇੰਟੀਗ੍ਰੇਸ਼ਨ
-   ਸ਼ਾਰਟਕੱਟਸ ਅਤੇ ਵਿਜਿਟਸ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਬਿਨਾਂ ਐਪ ਖੋਲ੍ਹੇ ਹੀ ਆਪਣਾ ਟੀਵੀ ਕੰਟਰੋਲ ਕਰੋ!
-   Headphones ਮੋਡ (ਜਿਸ ਨੂੰ private listening ਵੀ ਆਖਦੇ ਹਨ) Mac, iPad, iPhone, VisionOS ਅਤੇ Apple TV 'ਤੇ ਸਪੋਰਟ ਕਰਦਾ ਹੈ (ਆਪਣੇ ਡਿਵਾਇਸ ਰਾਹੀਂ TV ਤੋਂ ਆਡੀਓ ਚਲਾਓ)
-   ਐਪ ਖੋਲ੍ਹਦੇ ਹੀ ਆਪਣੀ ਲੋਕਲ ਨੈਟਵਰਕ 'ਤੇ ਡਿਵਾਇਸ ਖੋਜੋ
-   ਐਪਲ ਦੇ ਨੈਟਿਵ SwiftUI ਡਿਜ਼ਾਈਨ ਸਿਸਟਮ ਨਾਲ ਇੰਟੂਇਟਿਵ ਡਿਜ਼ਾਈਨ
-   ਤੇਜ਼ ਤੇ ਹਲਕਾ, ਹਰ ਡਿਵਾਇਸ 'ਤੇ 8MB ਤੋਂ ਘੱਟ ਤੇ ਅੱਧੇ ਸਕਿੰਟ ਤੋਂ ਵੀ ਘੱਟ ਸਮੇਂ 'ਚ ਖੁਲ ਜਾਂਦਾ ਹੈ!
-   ਖੁੱਲਾ ਸਰੋਤ (https://github.com/msdrigg/roam)

## ਆਮ ਸਮੱਸਿਆਵਾਂ

-   ਜੇ Roam ਮੇਰਾ TV ਆਪਣੇ ਆਪ ਨਹੀਂ ਲੱਭ ਰਿਹਾ ਤਾਂ ਕੀ ਕਰਾਂ
    -   [ਇਥੇ ਵੇਖੋ](/manually-add-tv)
-   ਮੇਰੇ Apple Watch 'ਤੇ Roam ਠੀਕ ਨਹੀਂ ਚੱਲ ਰਿਹਾ
    -   ਮਿਹਰਬਾਨੀ ਕਰਕੇ **Settings -> System -> Advanced System Settings -> Control by mobile apps** 'ਚ ਜਾਓ ਅਤੇ ਇਹ ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਇਹ **Permissive** 'ਤੇ ਸੈੱਟ ਹੈ
-   ਮੇਰੇ TV 'ਤੇ headphones mode (private listening) ਕਿਉਂ ਨਹੀਂ ਚੱਲਦਾ?
    -   Headphones mode ਕੁਝ TV 'ਤੇ ਹੁਣੇ ਕੰਮ ਨਹੀਂ ਕਰ ਰਿਹਾ। ਜੇ Roam ਨਾਲ headphones mode ਨਹੀਂ ਚੱਲਦਾ, ਪਰ official Roku app ਨਾਲ ਚੱਲਦਾ ਹੈ, ਤਾਂ ਆਪਣੀ Roku ਦਾ ਮਾਡਲ ਤੇ ਹੋਰ ਲੋੜੀਂਦੀ ਜਾਣਕਾਰੀ [roam-support@msd3.io](mailto:roam-support@msd3.io) 'ਤੇ ਈਮੇਲ ਕਰੋ। ਤੁਹਾਡੀ ਰਿਪੋਰਟ ਮਦਦ ਕਰੇਗੀ ਕਿ ਮੈਂ ਇਸ ਬੱਗ ਨੂੰ ਠੀਕ ਕਰਨ ਲਈ ਕਿੱਥੇ ਵੇਖਾਂ।
-   ਜੇ ਮੇਰੇ ਕੋਲ ਹੋਰ ਸਮੱਸਿਆ ਜਾਂ ਫੀਡਬੈਕ ਦੇਣਾ ਹੋਵੇ ਤਾਂ?
    -   ਜੇ ਇਹ ਬੱਗ ਹੈ, ਵਧੀਆ ਇਹ ਹੋਵੇਗਾ ਕਿ ਐਪ ਵਿਚੋਂ ਹੀ ਫੀਡਬੈਕ ਰਿਪੋਰਟ ਜਮ੍ਹਾਂ ਕਰਵਾਈ ਜਾਵੇ
        -   Roam ਐਪ ਖੋਲ੍ਹੋ ਤੇ ਸੈਟਿੰਗ ਪੇਜ 'ਤੇ ਜਾਓ
        -   "Send feedback" 'ਤੇ ਕਲਿਕ ਕਰੋ। ਇਹ ਇਕ ਡਾਇਗਨੋਸਟਿਕ ਰਿਪੋਰਟ ਬਣਾਏਗਾ ਜਿਸਨੂੰ roam support (roam-support@msd3.io) ਨੂੰ ਭੇਜ ਸਕਦੇ ਹੋ
        -   ਜੇ ਤੁਹਾਡਾ ਐਪ ਕਰੈਸ਼ ਹੋ ਰਿਹਾ ਹੈ, ਤਾਂ ਇਹ ਵੀ ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਸੈਟਿੰਗ -> Privacy & Security -> Analytics & Improvements 'ਚ ਤੁਹਾਡੀ analytics ਚਾਲੂ ਹੋਣ
            -   "Share iPhone & Watch Analytics" ਤੇ "Share With App Developers" ਚਾਲੂ ਕਰੋ ਤਾਂ ਜੋ Apple ਤੁਹਾਡੇ ਐਪ ਕਰੈਸ਼ ਹੋਣ 'ਤੇ ਮੈਨੂੰ ਰਿਪੋਰਟ ਕਰ ਸਕੇ
    -   ਜੇ ਇਹ ਨਵੀਂ ਖੂਬੀ ਦੀ ਮੰਗ ਹੈ, ਤਾਂ ਤੁਸੀਂ ਈਮੇਲ ਕਰ ਸਕਦੇ ਹੋ (roam-support@msd3.io), Roam ਐਪ 'ਚ ਸਿੱਧਾ ਗੱਲ ਕਰ ਸਕਦੇ ਹੋ (Settings -> Chat with the Developer) ਜਾਂ [Roam Discord](https://discord.gg/FqaTNRccbG) 'ਤੇ ਸ਼ਾਮਲ ਹੋ ਸਕਦੇ ਹੋ।
-   ਕਈ ਵਾਰ iPad 'ਤੇ arrow keys ਕਿਉਂ ਕੰਮ ਨਹੀਂ ਕਰਦੀਆਂ?
    -   ਇਹ ਇਸਕਰਕੇ ਹੁੰਦਾ ਹੈ ਕਿਉਂਕਿ ਕਈ ਵਾਰ iPadOS arrow keys ਨੂੰ ਆਪਣੀ ਸਕਰੀਨ ਨੇਵੀਗੇਸ਼ਨ ਲਈ ਵਰਤ ਲੈਂਦਾ ਹੈ, ਜਿਸ ਨਾਲ ਅਸੀਂ ਉਹਨਾਂ ਨੂੰ ਪਕੜ ਨਹੀਂ ਸਕਦੇ
    -   ਤੁਸੀਂ ਇਹ ਕੰਮ Settings -> Accessibility -> Keyboards 'ਚ "Full Keyboard Access" ਬੰਦ ਕਰਕੇ ਜਾਂ ਫਿਰ Settings -> Accessibility -> Keyboards -> Full Keyboard Access -> Commands -> Basic 'ਚ "Move Up", "Move Down", "Move Left" ਅਤੇ "Move Right" commands ਬੰਦ ਕਰਕੇ ਕਰ ਸਕਦੇ ਹੋ
-   ਮੇਰੀ ਕੁੰਜੀ ਪਡਾਈ TV 'ਤੇ ਕਿਉਂ ਨਹੀਂ ਦਿਖਦੀ
    -   ਕੁਝ Roku ਐਪਸ ਵਿਚ ਐਪ ਹਾਰਡਵੇਅਰ ਕੀਬੋਰਡ ਇੰਪੁੱਟ ਨੂੰ ਅਣਡਿੱਠਾ ਕਰ ਦਿੰਦੇ ਹਨ। ਤੁਸੀਂ ਇਸਨੂੰ ਪਰਖ ਸਕਦੇ ਹੋ ਕਿ ਇਹ Roam ਦੀ ਸਮੱਸਿਆ ਹੈ ਜਾਂ ਐਪ ਦੀ, ਕਿ official Roku App ਵਿਚ ਕੀਬੋਰਡ ਇੰਟਰੀ ਵਰਤ ਕੇ ਦੇਖ ਲਵੋ ਕਿ ਓਥੇ ਕੰਮ ਕਰਦੀ ਹੈ ਜਾਂ ਨਹੀਂ
    -   ਉਹ ਐਪਸ ਜਿਨ੍ਹਾਂ 'ਚ ਇਹ ਬੱਗ ਹੈ
        -   Prime Video
-   ਮੇਰੇ iPhone ਤੇ Mac ਤੇ Roam ਚੱਲ ਜਾਂਦੀ ਹੈ ਪਰ Apple Watch 'ਤੇ ਕਿਉਂ ਨਹੀਂ?
    -   WatchOS ਐਪ TV ਨਾਲ ਇਸਦੇ ECP API ਰਾਹੀਂ ਜੁੜਦੀ ਹੈ, ਜੋ ਕਿ ਕੁਝ Roku TV 'ਤੇ ਯਕੀਨੀ ਤੌਰ 'ਤੇ ਚਾਲੂ ਹੋਣੀ ਚਾਹੀਦੀ ਹੈ। ਇਹ ਚਾਲੂ ਕਰਨ ਲਈ, **Settings -> System -> Advanced System Settings -> Control by mobile apps** 'ਤੇ ਜਾਓ ਅਤੇ "Network Access" ਨੂੰ "Permissive" 'ਤੇ ਸੈੱਟ ਕਰੋ
