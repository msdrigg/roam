# TV ਨੂੰ ਹੱਥੋਂ ਐੱਡ ਕਰਨਾ

1. ਆਪਣੇ TV ਦਾ IP ਐਡਰੈੱਸ ਲੱਭੋ
    - ਆਪਣਾ TV ਚਾਲੂ ਕਰੋ ਅਤੇ **Settings** > **Network** > **About** 'ਤੇ ਜਾਓ
    - IP ਐਡਰੈੱਸ ਐਸ ਤਰ੍ਹਾਂ ਹੋ ਸਕਦਾ ਹੈ: 10.x.x.x, 172.x.x.x, 173.x.x.x ਜਾਂ 192.168.x.x
    - ਇਸ ਸਫ਼ੇ 'ਤੇ "Gateway" ਐਡਰੈੱਸ ਅਤੇ "IP Address" ਦੱਸਿਆ ਹੋ ਸਕਦਾ ਹੈ। ਯਕੀਨੀ ਬਣਾਓ ਤੁਸੀਂ "Gateway" ਐਡਰੈੱਸ ਨਹੀਂ ਵਰਤ ਰਹੇ
2. Roam ਸੈਟਿੰਗਸ 'ਤੇ ਜਾਓ ਅਤੇ "Add a device manually" 'ਤੇ ਕਲਿੱਕ ਕਰੋ
3. ਆਪਣੇ ਡਿਵਾਈਸ ਨੂੰ ਜੋ ਮਰਜ਼ੀ ਨਾਂ ਦਿਓ ਅਤੇ TV 'ਤੇ ਦਿਖਾਇਆ ਗਿਆ IP ਐਡਰੈੱਸ ਠੀਕ ਠੀਕ ਰੂਪ ਵਿੱਚ ਭਰੋ
4. Save 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਹੁਣ ਤੁਹਾਡਾ Roku ਜੁੜ ਸਕੇਗਾ ਅਤੇ ਆਮ ਵਾਂਗ ਕੰਮ ਕਰੇਗਾ

## ਜੇ ਤੁਸੀਂ TV ਹੱਥੋਂ ਐੱਡ ਕਰ ਲਓ ਫਿਰ ਵੀ Roam ਕਨੈਕਟ ਨਹੀਂ ਹੁੰਦਾ ਜਾਂ ਕਨੈਕਸ਼ਨ ਠੀਕ ਕੰਮ ਨਹੀਂ ਕਰ ਰਿਹਾ?

ਜੇ Roam ਫਿਰ ਵੀ ਤੁਹਾਡਾ Roku ਕੰਟਰੋਲ ਨਹੀਂ ਕਰ ਸਕਦਾ ਤਾਂ ਇਹ ਹਦਾਇਤਾਂ ਅਜ਼ਮਾਓ

-   [WatchOS ਸਿਰਫ]: **Settings -> System -> Advanced System Settings -> Control by mobile apps** ਤੇ ਜਾਓ ਅਤੇ ਦੇਖੋ ਇਹ **Permissive** 'ਤੇ ਸੈੱਟ ਹੈ
-   ਯਕੀਨੀ ਬਣਾਓ ਤੁਹਾਡਾ iOS ਡਿਵਾਈਸ ਉਸੇ WiFi ਨੈੱਟਵਰਕ 'ਤੇ ਜੁੜਿਆ ਹੋਇਆ ਹੈ ਜਿਸ ਤੇ ਤੁਹਾਡਾ Roku TV ਹੈ
-   ਯਕੀਨੀ ਬਣਾਓ ਤੁਹਾਡਾ TV ਚਾਲੂ ਹੈ
-   ਯਕੀਨੀ ਬਣਾਓ Roam ਲਈ Local Network Permissions ਐਨੇਬਲ ਹਨ (ਜੇ ਤੇ ਪਹਿਲਾਂ ਤੋਂ ਐਨੇਬਲ ਹਨ ਤਾਂ ਡਿਸੇਬਲ ਕਰ ਕੇ ਮੁੜ ਐਨੇਬਲ ਕਰੋ)
    -   macOS 'ਤੇ: System Settings -> Privacy and Security -> Local Network -> Roam ਤੇ ਜਾਓ
    -   iOS 'ਤੇ: Settings -> Apps -> Roam -> Local Network ਤੇ ਜਾਓ
-   ਹੋਰ ਸੰਭਾਵਾਵਾਂ ਲਈ ਇੱਥੇ ਵੇਖੋ [https://support.roku.com/article/115001480188](https://support.roku.com/article/115001480188)

## ਜੇ ਮੇਰੇ ਕੋਲ ਜਟਿਲ ਨੈੱਟਵਰਕ/VPN ਸੈਟਅੱਪ ਹੈ? ਇਹ ਐਪ ਕਿਹੜੇ ਪਰੋਟੋਕੌਲ ਵਰਤਦੀ ਹੈ?

-   Roam ਤੁਹਾਡੇ TV ਨਾਲ ਸੰਪਰਕ ਕਰਨ ਲਈ ਕਈ ਵੱਖ-ਵੱਖ ਪਰੋਟੋਕੌਲ ਵਰਤਦੀ ਹੈ
    -   TCP (HTTP/Websockets) ਪੋਰਟ 8060 'ਤੇ TV ਨੂੰ ਕਮਾਂਡ ਭੇਜਣ ਤੇ ਡਿਵਾਈਸ ਸਟੇਟ ਪੁੱਛਣ ਲਈ
    -   WOL ਮੈਜਿਕ ਪੈਕੇਟ (UDP multicast ਐਡਰੈੱਸ 255.255.255.255 'ਤੇ) TV ਨੂੰ ਡੀਪ ਸਲੀਪ ਤੋਂ ਜਗਾਉਣ ਲਈ
    -   RDP (UDP) ਪੋਰਟ 6970 'ਤੇ ਹੈੱਡਫੋਨ ਮੋਡ ਆਡੀਓ ਸਟ੍ਰੀਮ ਲਈ
-   ਸਭ ਤੋਂ ਵੱਧ Roku TV's ਪੋਰਟ 8060 ਵਰਦੀਆਂ ਹਨ ਅਤੇ TV ਪਾਸੇ ਤੋਂ ਇਹ ਨਹੀਂ ਬਦਲੀ ਜਾ ਸਕਦੀ। ਪਰ ਜੇ ਤੁਹਾਡਾ ਕੋਈ ਪੋਰਟ ਫਾਰਵਰਡਿੰਗ ਸੈਟਅੱਪ ਹੈ ਅਤੇ ਤੁਸੀਂ Roam ਤੋਂ ਵੱਖਰਾ ਆਉਟਗੋਇੰਗ ਪੋਰਟ ਵਰਤਣਾ ਚਾਹੁੰਦੇ ਹੋ, ਤਾਂ ਇਹ ਸੰਭਵ ਹੈ। ਤੁਸੀਂ ਸਿਰਫ "Ip Address" ਫੀਲਡ ਵਿੱਚ `[IP]:[Port]` ਲਿਖੋ ਨਾ ਕਿ ਸਿਰਫ `[IP]`। ਉਦਾਹਰਨ ਵਜੋਂ, `192.168.8.242:8061` ਭਰੋ ਤਾਂ ਪੋਰਟ `8061` ਵਰਤੀ ਜਾਏਗੀ।
