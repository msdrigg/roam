# ਟੀਵੀ ਹੱਥੋਂ ਕਿਵੇਂ ਸ਼ਾਮਲ ਕਰੀਏ

1. ਆਪਣੇ ਟੀਵੀ ਦਾ IP ਐਡਰੈੱਸ ਲੱਭੋ
    - ਟੀਵੀ ਆਨ ਕਰੋ ਅਤੇ **Settings** > **Network** > **About** ਉੱਤੇ ਜਾਓ
    - ਜੇਕਰ ਤੁਸੀਂ ਕੋਲ ਭੌਤਿਕ ਰੀਮੋਟ ਜਾਂ ਹੋਰ ਕੋਈ ਟੀਵੀ ਕੰਟਰੋਲ ਕਰਨ ਦਾ ਤਰੀਕਾ ਨਹੀਂ ਹੈ, ਤਾਂ ਆਪਣੇ ਘਰ ਦੇ ਰਾਊਟਰ ਦੇ ਐਡਮਿਨ ਇੰਟਰਫੇਸ ਜਾਂ DHCP ਕਲਾਇੰਟ ਲਿਸਟ ਵਿੱਚ ਰੋਕੂ ਦਾ IP ਐਡਰੈੱਸ ਲੱਭੋ
    - IP ਐਡਰੈੱਸ ਕੁਝ ਇਹ ਜਿਹਾ ਹੋ ਸਕਦਾ ਹੈ: 10.x.x.x, 172.x.x.x, 173.x.x.x ਜਾਂ 192.168.x.x
    - ਇਸ ਪੇਜ ਤੇ "Gateway" ਐਡਰੈੱਸ ਅਤੇ "IP Address" ਦੋਵੇਂ ਹੋ ਸਕਦੇ ਨੇ। ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਸੀਂ "Gateway" ਐਡਰੈੱਸ ਨਾ ਵਰਤ ਰਹੇ ਹੋ
2. Roam ਸੈਟਿੰਗਸ ਵਿੱਚ ਜਾਓ ਅਤੇ "Add a device manually" ਉੱਤੇ ਕਲਿੱਕ ਕਰੋ
3. ਆਪਣੇ ਡਿਵਾਈਸ ਨੂੰ ਜੋ ਵੀ ਨਾਂ ਦੇਣਾ ਹੈ ਦੇਵੋ, ਅਤੇ ਡਿਵਾਈਸ ਦਾ IP ਬਿਲਕੁਲ ਓਨਜ ਐਡ ਕਰੋ ਜਿਵੇਂ ਕਿ ਤੁਹਾਡੇ Roku ਟੀਵੀ ਉੱਤੇ ਦਿਖਾਇਆ ਗਿਆ ਹੈ
4. Save ਉੱਤੇ ਕਲਿੱਕ ਕਰੋ। ਹੁਣ ਤੁਹਾਡਾ ਰੋਕੂ ਕਨੈਕਟ ਹੋ ਸਕਦਾ ਅਤੇ ਨਾਰਮਲ ਤਰ੍ਹਾਂ ਕੰਮ ਕਰੇਗਾ

## ਜੇਕਰ ਤੁਸੀਂ ਟੀਵੀ ਹੱਥੋਂ ਆਡ ਕਰਲੀ ਤੇ Roam ਫਿਰ ਵੀ ਕਨੈਕਟ ਨਹੀਂ ਕਰ ਸਕਦਾ ਜਾਂ ਕੁਨੈਕਸ਼ਨ ਠੀਕ ਨਹੀਂ ਚਲ ਰਿਹਾ?

ਜੇ Roam ਫਿਰ ਵੀ ਤੁਹਾਡੇ ਰੋਕੂ ਨੂੰ ਕੰਟਰੋਲ ਨਹੀਂ ਕਰ ਸਕਦਾ, ਤਾਂ ਇਹ ਹਦਾਇਤਾਂ ਅਜ਼ਮਾਓ

-   [WatchOS ਕੇਵਲ]: **Settings -> System -> Advanced System Settings -> Control by mobile apps** ਉੱਤੇ ਜਾਕੇ ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਇਹ **Permissive** ਤੇ ਸੈਟ ਹੈ
-   ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਹਾਡਾ iOS ਡਿਵਾਈਸ ਉਨ੍ਹਾਂ ਹੀ WiFi ਨੈੱਟਵਰਕ ਤੇ ਕਨੈਕਟ ਹੈ ਜਿਸਤੇ ਤੁਹਾਡਾ Roku TV ਵੀ ਝੁੜਿਆ ਹੋਇਆ
-   ਯਕੀਨੀ ਬਣਾਓ ਤੁਹਾਡਾ ਟੀਵੀ ਆਨ ਹੈ
-   ਯਕੀਨੀ ਬਣਾਓ Roam ਲਈ Local Network Permissions ਐਨੇਬਲ ਹਨ (ਜਾਂ ਜੇ ਪਹਿਲਾਂ ਦੇ ਹੋਏ ਹਨ ਤਾਂ ਡਿਸੇਬਲ ਕਰਕੇ ਫੇਰ ਐਨੇਬਲ ਕਰੋ)
    -   macOS ਤੇ: System Settings -> Privacy and Security -> Local Network -> Roam
    -   iOS ਤੇ: Settings -> Apps -> Roam -> Local Network
-   ਜੇਕਰ ਤੁਹਾਡੇ ਘਰ ਦੇ ਨੈੱਟਵਰਕ ਵਿੱਚ ਕੋਈ ਬਦਲਾਅ ਆਇਆ ਤੇ ਪੁਰਾਣਾ ਚਲ ਰਿਹਾ ਡਿਵਾਈਸ ਕੰਮ ਕਰਨਾ ਬੰਦ ਹੋ ਗਿਆ, ਤਾਂ Roam ਵਿੱਚੋਂ ਉਸ ਡਿਵਾਈਸ ਨੂੰ ਹਟਾ ਕੇ ਦੁਬਾਰਾ ਸਕੈਨ ਕਰੋ
-   ਜੇਕਰ ਰੋਕੂ WiFi ਤੇ ਕਨੈਕਟ ਨਹੀਂ ਅਤੇ ਤੁਹਾਡੇ ਕੋਲ ਭੌਤਿਕ ਰੀਮੋਟ ਨਹੀਂ, ਤਾਂ Roku ਦੀ ਮੋਬਾਈਲ ਐਪ ਨਾਲ ਕਨੈਕਸ਼ਨ ਬਣਾਉਣ ਦੇ ਹਦਾਇਤਾਂ ਇਥੇ ਵੇਖੋ: [https://support.roku.com/article/115001480188](https://support.roku.com/article/115001480188)
-   ਹੋਰ ਸੰਭਾਵਨਾਵਾਂ ਇਥੇ ਵੇਖੋ [https://support.roku.com/article/115001480188](https://support.roku.com/article/115001480188)

## ਜੇ ਮੇਰੇ ਕੋਲ ਕਿਸੇ ਤਰ੍ਹਾਂ ਦੀ ਜਟਿਲ ਨੈੱਟਵਰਕ/VPN ਸੈਟਅੱਪ ਹੈ? ਇਹ ਐਪ ਕਿਹੜੇ ਪਰੋਟੋਕੋਲ ਵਰਤਦੀ ਹੈ?

-   Roam ਟੀਵੀ ਨਾਲ ਸੰਚਾਰ ਕਰਨ ਲਈ ਕਈ ਵੱਖ-ਵੱਖ ਪਰੋਟੋਕੋਲ ਵਰਤਦੀ ਹੈ
    -   TCP (HTTP/Websockets) ਪੋਰਟ 8060 ਉੱਤੇ TV ਨੂੰ ਕਮਾਂਡ ਭੇਜਣ ਅਤੇ ਡਿਵਾਈਸ ਦੀ ਹਾਲਤ ਦੇਖਣ ਲਈ
    -   WOL ਮੈਜਿਕ ਪੈਕੇਟ (UDP multicast 255.255.255.255 ਤੇ) TV ਨੂੰ ਡੀਪ ਸਲੀਪ ਤੋਂ ਜਗਾਉਣ ਲਈ
    -   RDP (UDP) ਪੋਰਟ 6970 ਉੱਤੇ ਹੈੱਡਫੋਨ ਮੋਡ ਆਡੀਓ ਸਟਰੀਮ ਲਈ
-   ਸਾਰੇ Roku TV ਪੋਰਟ 8060 ਵਰਤਦੇ ਹਨ ਅਤੇ ਇਹ ਪੋਰਟ ਟੀਵੀ ਪਾਸੇ ਬਦਲੀ ਨਹੀਂ ਜਾ ਸਕਦੀ। ਪਰ ਜੇਕਰ ਤੁਸੀਂ ਕੁਝ ਪੋਰਟ ਫਾਰਵਰਡਿੰਗ ਕਰ ਰਹੇ ਹੋ ਅਤੇ Roam ਵਿੱਚੋਂ ਵੱਖਰਾ ਆਉਟਗੋਇੰਗ ਪੋਰਟ ਵਰਤਣਾ ਚਾਹੁੰਦੇ ਹੋ, ਤਾਂ ਇਹ ਸੰਭਵ ਹੈ। ਤੁਹਾਨੂੰ "Ip Address" ਫੀਲਡ ਵਿੱਚ ਸਿਰਫ਼ [IP] ਦੀ ਥਾਂ [IP]:[Port] ਇੰਟਰ ਕਰਨਾ ਪਵੇਗਾ। ਉਦਾਹਰਣ ਵਜੋਂ, `192.168.8.242:8061` ਇੰਟਰ ਕਰੋ ਅਤੇ ਪੋਰਟ `8061` ਵਰਤੀ ਜਾਵੇਗੀ।
