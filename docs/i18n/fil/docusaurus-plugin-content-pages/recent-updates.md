# Roam Roadmap

## Kamakailang Mga Update

-   Nagdagdag ng mga kontrol na widget: Maglaro, I-mute, Palitan ang Volume at Pumili mula sa Control center!
-   Nagdagdag ng mas mahusay na paghawak sa text field para sa maraming mga roku apps
    -   Auto-bukas ang text field kapag available ang pag-edit ng text
    -   Kopyahin, Putulin, I-paste mula sa macOS (gamit ang keyboard)
    -   Kopyahin, Putulin, I-paste + Generalized edit sa iOS
-   Mas maayos na pag-uulat tungkol sa mga pahintulot at konektibidad ng local network
-   Pinabuting pag-andar ng keyboard
-   Mga pagpapabuti sa koneksyon ng estabilidad
-   Inayos ang mga widget at mga shortcut sa macOS 15
